Detailed Description

Quadrature Encoder library for RP2040/RP2350.

Author
Giovanni di Dio Bruno
Version
0.2.0

This library can be used with any RP240/RP2350 board, such as Arduino Nano RP2040 Connect, RPI Pico and RPI Pico 2.

PioEncoder::PioEncoder ( const uint8_t pin,

PIO pio = pio0,
const uint sm = -1,
const int max_step_rate = 0 )

Constructor for a quadrature encoder PioEncoder object. The pins for Channels A and B must be consecutive, and the pin for Channel A must be the smaller of the two pin values.

Parameters
pinThe pin for Channel A of the encoder.
resolution(Optional) The resolution of the encoder (i.e., the number of encoder counts per one revolution). The default value is 1 count per revolution.
flip(Optional) Whether to flip the encoder or not. The default value is false.
zero_offset(Optional) The initial amount of offset the encoder counts by. Defaulted to 0.
count_mode(Optional) The counting mode for the encoder. Must be one of COUNT_1X, COUNT_2X, or COUNT_4X.
pio(Optional) Internal #define in the pio.h file. Defaulted to pio0.
sm(Optional) Default value of -1. Probably don't want to mess with this.
max_step_rate(Optional) The maximum stepping rate. Defaulted to 0.
