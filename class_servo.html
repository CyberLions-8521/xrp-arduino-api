Detailed Description

Allows Arduino boards to control a variety of servo motors.

Author
Michael Margolis, Arduino
Version
1.2.2

This library can control a great number of servos.
It makes careful use of timers: the library can control 12 servos using only 1 timer.
On the Arduino Due you can control up to 60 servos.

NOTE: For FRC purposes, the stm32f4 and xmc directories from the original Github repository have been removed.

Attach the given pin to the next free channel, and sets pinMode. Also sets min and max values (in microseconds) for writes. The default minimum is 544, and the default maximum is 2400.
